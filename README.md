# Bài tập lớn nhập môn trí tuệ nhân tạo 
- Ứng dụng thuật toán IDA* giải trò chơi sokoban
# Công cụ
- Thư viện Pygame
```
pip install pygame
```

## Chạy chương trình
```bash
python main.py
```

## Điều khiển
- `LEFT/RIGHT`: đổi level
- `ENTER`: chạy solver IDA* (push-based)
- `P`: chơi thủ công
- `R`: reset level
- `Q`: thoát
